◆ WorldTime

WorldTime TimeManager.WorldTime
getset

Gets or sets the world time.

Assigning to WorldTime will adjust GameTime appropriately, however, using the Set and Add methods directly on WorldTime will not work. To adjust WorldTime assign a new WorldTime, or assign immediately after adjusting.

Example: WorldTime = WorldTime.SetHour(8).SetMinute(0) will properly change the time to 8:00, leaving everything else the same.

The world time.
